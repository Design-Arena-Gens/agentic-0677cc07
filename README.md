## Hi Designer

Create quick, personalized greetings in the browser. Adjust the tone, sprinkle in emojis, switch fonts, and add a custom note—perfect for brightening someone’s day with a thoughtful “hi.”

### Preview

- Hero introduction with context for the tool
- Interactive greeting designer:
  - Name input
  - Tone chips with optional auto-cycling
  - Emoji toggle
  - Font selector
  - Custom note field
- Live preview card that reflects selections in real time

### Running Locally

```bash
yarn install
yarn dev
```

Open http://localhost:3000 in your browser to use the designer.

### Tech

- Next.js App Router
- TypeScript + CSS Modules
- Geist font family via `next/font`

### Deploy

The project is ready for Vercel (`yarn build && yarn start`). Use your preferred workflow to ship it live.
